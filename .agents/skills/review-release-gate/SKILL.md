---
name: review-release-gate
description: Perform a read-only release or milestone review across correctness, scientific invariants, conformance, architecture, security, and documentation.
---

Run independent reviews with the scientific invariant, conformance, .NET architecture, test, and relevant extension or model-governance agents.

Consolidate findings without duplicating them. Report blocking, important, and minor issues; missing negative tests; specification conflicts; migration risks; and a clear safe-to-merge verdict.

Verify release build, tests, formatting, architecture checks, conformance fixtures, offline CI behavior, source locks, changelog, and gate documentation. Do not edit files during the review pass.
