# Post-FE-09 Whole-Project Integrity Remediation Gate

Status: Accepted

Date: 2026-07-18

Authority: ADR 0045

## Goal

Close the independently reproduced integrity, parser, public-contract, and
delivery-evidence defects without expanding scientific authority, provider
rights, PHP compatibility, FE-10, or FE-11 scope.

## Required Work

- centralize cross-platform Research Workspace path containment;
- revalidate source inputs under the commit lock;
- clear predecessor-bound downstream current pointers on successor authority;
- verify retained provider bytes on cache lookup and index rebuild;
- apply one provider secret-shaped-value policy to decoded names and values;
- fail closed for nested RIS blocks and duplicate Scopus headers;
- make canonical `WorkId` rendering round-trip while retaining ambiguous
  namespace-token rejection;
- reject default canonical scientific timestamps;
- round-trip superseded protocol authority while keeping active admission
  explicit;
- snapshot AI proposal values and harden Extensibility construction;
- resolve the pinned SDK, freeze an explicit mutation matrix, add ownership,
  and correct current-state claims.

## Required Negative Evidence

- Linux case-only sibling traversal and explicit parent traversal;
- source mutation between analysis and commit;
- stale Screening, Full Text, workflow, and reporting pointers after a
  Deduplication successor;
- missing and tampered cache bodies during lookup and rebuild;
- credential/contact-shaped Crossref query values;
- nested RIS and duplicate normalized Scopus headers;
- ambiguous namespace-prefixed WorkId values;
- default timestamps on construction and rehydration;
- mutable proposal inputs and undefined extension capabilities.

## Verification

1. Focused affected-project Release tests.
2. Architecture and conformance tests.
3. Explicit scientific-invariant mutation matrix.
4. `scripts/verify.ps1`.
5. Site distribution verification and `git diff --check`.
6. Independent scientific-invariant and extension-security review.

## Exit Criteria

- all reproduced failures have a focused regression test;
- no downstream predecessor-bound artifact is claimed current;
- no cache or provider descriptor can silently carry unverified or forbidden
  bytes;
- canonical construction and rehydration reject the same invalid state;
- full verification passes with zero warnings and no network dependency;
- completion evidence identifies any remote-only governance work not performed.
