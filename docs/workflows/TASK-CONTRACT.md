## Stop Conditions

Stop if:

- source-of-truth levels conflict
- behavior requires a new ADR
- fixture expectations are unclear
- implementation would resolve an open conflict implicitly
- tests require live provider or LLM access
- the requested change would weaken protocol immutability, human authority, provenance, or digest boundaries

## Explicit Non-Claims

Do not claim:

- blueprint conformance
- PHP compatibility
- protocol conformance
- bundle conformance
- scientific validity of a model output
- gate acceptance

unless the task explicitly permits the claim and cites the required evidence.
